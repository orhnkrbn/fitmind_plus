# 📌 Overview

**FitMind+**, fiziksel gelişim + mental motivasyonu bir araya getiren **yapay zekâ destekli bir kişisel koç** uygulamasıdır.  
Kullanıcıya bilimsel antrenman rehberi, doğru beslenme bilgisi, motivasyon artırıcı içerikler ve AI tabanlı kişisel yönlendirmeler sunar.

Uygulama modern, akıcı, Apple Fitness benzeri bir UI/UX ile geliştirilmiştir ve tamamen Flutter üzerinde ilerlemektedir.

---

# ✨ Features

### 🧠 **AI Content Engine**
- Local AI (Ollama – Qwen/phi modelleri)  
- Cloud AI (OpenAI / Anthropic / DeepSeek) – Hazır altyapı  
- Akıllı içerik üretimi  
- FitMind yönlendirme sistemi

### 💪 **Training Zone**
- Bilimsel antrenman içerikleri  
- Kas grubu bazlı eğitim  
- JSON tabanlı dinamik içerik sistemi

### 🥗 **Nutrition Notes**
- Basit & doğru beslenme prensipleri  
- Temel makro/mikro öğeler  
- AI’ya göre kişisel öneri desteği

### ⚡ **Modern UI / Smooth Animations**
- iOS/Apple Fitness tarzı minimal tasarım  
- Altın oran gözetilmiş layout  
- High performance Flutter widget mimarisi

### 🎯 **Motivation Hub**
- Günlük motivasyon kartları  
- AI’dan kişisel yönlendirmeler  
- Odak modları
