# FitMind+ — AI destekli fitness ve motivasyon koçu

Kısa amaç: Sade, estetik ve hızlı bir Flutter uygulaması. Riverpod (state), go_router (navigasyon),
freezed/json_serializable (model kod üretimi), intl (i18n) ve sıkı lint kuralları kullanılır.

Modüller:
- app/: uygulama başlangıcı, tema, router
- core/: sabitler, tema yardımcıları, ortak util'ler
- features/: özellik bazlı modüller (onboarding, auth, home, workouts, progress, motivation, nutrition)
